PRACTICAL 1(A) – TO IMPLEMENT ALL LOGIC GATES IN LOGISIM SIMULATOR

 

 

BASIC LOGIC GATE: A logic gate is a basic building block of a digital circuit that has two inputs and one output.

 

·       These gates are implemented using electronic switches like transistors, diodes.

·       The basic logic gates are classified into seven types: AND gate, OR gate, XOR gate, NAND gate, NOR gate, XNOR gate, and NOT gate.

 

 

 

·       All the logic gates have two inputs except the NOT gate, which has only one input.

·       The truth table is used to show the logic gate function

·       When drawing a truth table, the binary values 0 and 1 are used

·       Every possible combination depends on the number of inputs.

 

 

1.NOT Gate (Inverter): This gate has one input and one output. It simply inverts the input; if the input is 0, the output is 1, and vice versa.

 

 

2.AND Gate: This gate has two or more inputs and one output. The output is high (1) only when all of the inputs are high (1).

 

 

3.OR Gate: This gate, like the AND gate, has two or more inputs and one output. The output is high (1) if at least one of the inputs is high (1).

 

 

 

 

4.NAND Gate: This gate is like an AND gate followed by a NOT gate. It has two or more inputs and one output. The output is low (0) only when all of the inputs are high (1).

 

 

 

 

 

 

5.NOR Gate: This gate is like an OR gate followed by a NOT gate. It has two or more inputs and one output. The output is low (0) if at least one of the inputs is high (1).

 

 

 

 

 

 



 

 

 

 

 

6.XOR Gate (Exclusive OR): This gate has two inputs and one output. The output is high (1) if the number of high inputs is odd.

 

7.XNOR Gate (Exclusive NOR): This gate has two inputs and one output. The output is high (1) if the number of high inputs is even.
